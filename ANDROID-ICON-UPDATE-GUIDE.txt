============================================
METHOD 1: AUTOMATIC (EASIEST)
============================================

When you rebuild the APK, Capacitor will
automatically sync your PWA icons to Android!

Steps:
------
1. BUILD-COMPLETE-APK-ALL-FIXES.bat
2. Wait for build to complete
3. Install APK
4. Check app icon → Should be earth logo! ✓

Why it works:
-------------
- Capacitor reads public/manifest.json
- Finds icon: /logo earth.png
- Copies to Android mipmap folders
- Generates all sizes automatically

If icon doesn't update:
-----------------------
Run: npx cap sync android
(Forces icon sync)

============================================
METHOD 2: ONLINE TOOL (EASY)
============================================

Use icon generator website:

Steps:
------
1. Go to: https://icon.kitchen/

2. Upload: public/logo earth.png

3. Configure:
   - Type: Launcher icon
   - Shape: Circle (for modern Android)
   - Background: Transparent or #1e40af (blue)

4. Download icon pack

5. Extract and copy to:
   android/app/src/main/res/
   (Replace existing mipmap folders)

6. Rebuild APK

============================================
METHOD 3: ANDROID STUDIO
============================================

If you have Android Studio:

Steps:
------
1. Open Android Studio

2. File → Open
   Select: android folder

3. Right-click: res folder

4. New → Image Asset

5. Configure:
   - Icon Type: Launcher Icons
   - Asset Type: Image
   - Path: Browse to public/logo earth.png
   - Shape: Circle
   - Background: Choose color or transparent

6. Click Next → Finish

7. Android Studio generates all sizes!

8. Close Android Studio

9. Rebuild APK

============================================
ANDROID ICON SIZES NEEDED:
============================================

Android requires multiple icon sizes:

mipmap-mdpi/      → 48x48 px
mipmap-hdpi/      → 72x72 px
mipmap-xhdpi/     → 96x96 px
mipmap-xxhdpi/    → 144x144 px
mipmap-xxxhdpi/   → 192x192 px

Files in each:
- ic_launcher.png (regular icon)
- ic_launcher_round.png (round icon)
- ic_launcher_foreground.png (adaptive icon layer)

============================================
TROUBLESHOOTING:
============================================

Icon doesn't change:
--------------------
1. Clear app data:
   Settings → Apps → QMAZ Project Map → Clear Data

2. Uninstall completely:
   Hold icon → Uninstall

3. Restart device

4. Install fresh APK

5. Icon should update!

Icon looks blurry:
------------------
- Use higher resolution source (512x512 or larger)
- Use PNG format (not JPG)
- Use icon generator for proper sizing

Icon has white background:
--------------------------
- Use transparent PNG
- Or set background color in icon generator
- Regenerate icons
