============================================
STEP 1: OPEN ANDROID STUDIO
============================================

1. Open Android Studio application

2. If you see "Welcome" screen:
   → Click "Open"

3. If you see a project already open:
   → File → Open

4. Browse to:
   C:\Users\edujk\Desktop\project map\acu-project-map-123456-789\android

5. Click "OK"

6. Wait for project to open and sync
   (This may take 2-5 minutes first time)

7. Let it download any needed components

============================================
STEP 2: LOCATE RES FOLDER
============================================

In Android Studio, left panel:

1. Look for "Project" panel (left side)

2. Make sure it shows "Android" view
   (dropdown at top of panel)

3. Expand: app → res

4. You should see:
   - drawable
   - mipmap-anydpi-v26
   - mipmap-hdpi
   - mipmap-mdpi
   - mipmap-xhdpi
   - mipmap-xxhdpi
   - mipmap-xxxhdpi
   - values

============================================
STEP 3: OPEN IMAGE ASSET STUDIO
============================================

1. Right-click on "res" folder

2. Select: New → Image Asset

3. A new window opens:
   "Asset Studio"

This is Android's official icon generator!

============================================
STEP 4: CONFIGURE ICON SETTINGS
============================================

In Asset Studio window:

Icon Type:
----------
✓ Select: "Launcher Icons (Adaptive and Legacy)"
  (Should be default)

Name:
-----
✓ Keep as: ic_launcher
  (Don't change this!)

Foreground Layer:
-----------------
1. Asset Type: Image

2. Path: Click folder icon 📁

3. Browse to:
   C:\Users\edujk\Desktop\project map\acu-project-map-123456-789\public

4. Select: logo earth.png

5. Click "Open"

6. Earth logo appears in preview!

Resize:
-------
✓ Adjust slider: 50-100%
  (100% = logo fills icon)
  (50% = logo has padding)
  
Recommended: 80-90%

Trim:
-----
✓ Check: "Yes" (removes transparent edges)

Background Layer:
-----------------
Option A: Color
  → Pick: #1e40af (blue)

Option B: Color
  → Pick: #FFFFFF (white)

Option C: Color
  → Pick: #000000 (black)

Recommended: #1e40af (matches your theme)

Legacy:
-------
Icon Shape:
✓ Select: "Circle"
  (Modern Android style)

============================================
STEP 5: PREVIEW ICONS
============================================

On the right side, you'll see previews:

Check different sizes:
---------------------
□ xxxhdpi (192x192) - looks good?
□ xxhdpi (144x144) - looks good?
□ xhdpi (96x96) - looks good?
□ hdpi (72x72) - looks good?
□ mdpi (48x48) - looks good?

Check different Android versions:
---------------------------------
□ Android 8.0+ (Adaptive) - looks good?
□ Android 7.1 (Round) - looks good?
□ Android 7.0 (Square) - looks good?

Check different backgrounds:
---------------------------
□ On white - looks good?
□ On black - looks good?
□ On colored - looks good?

If NOT good:
-----------
→ Go back to Step 4
→ Adjust Resize slider
→ Change background color
→ Check preview again

============================================
STEP 6: GENERATE ICONS
============================================

When preview looks good:

1. Click "Next" button (bottom right)

2. New screen shows:
   "Confirm Icon Path"

3. Shows list of files that will be created:
   - res/mipmap-mdpi/ic_launcher.png
   - res/mipmap-mdpi/ic_launcher_round.png
   - res/mipmap-hdpi/ic_launcher.png
   - res/mipmap-hdpi/ic_launcher_round.png
   - ... (more sizes)

4. Click "Finish"

5. Android Studio generates all icon files!

6. You'll see:
   "Image Asset Studio - Operation complete"

7. Click "OK"

============================================
STEP 7: VERIFY ICONS CREATED
============================================

In Android Studio, left panel:

1. Expand: res → mipmap-mdpi

2. You should see:
   - ic_launcher.png (NEW! earth logo)
   - ic_launcher_foreground.png (NEW!)
   - ic_launcher_round.png (NEW!)

3. Double-click ic_launcher.png

4. Earth logo should display! ✓

5. Check other mipmap folders too
   (All should have new earth logo icons)

============================================
STEP 8: CLOSE ANDROID STUDIO
============================================

1. File → Exit
   (Or close window)

2. If asks "Save changes?"
   → Click "Yes" or "Save All"

3. Android Studio closes

Icons are now in your project! ✓

============================================
STEP 9: BUILD APK
============================================

Back to your project folder:

1. Double-click: BUILD-COMPLETE-APK-ALL-FIXES.bat

2. Wait 10-15 minutes

3. APK created with earth logo icon!

============================================
STEP 10: INSTALL ON DEVICE
============================================

IMPORTANT - Follow exactly:

1. Uninstall old app completely
   Settings → Apps → QMAZ Project Map → Uninstall

2. RESTART your device
   (Power off and on - very important!)

3. Transfer new APK to device

4. Install APK

5. Check app drawer

6. Earth logo should appear! 🌍

============================================
❓ TROUBLESHOOTING:
============================================

Android Studio won't open project?
----------------------------------
→ Need to install Android SDK
→ Tools → SDK Manager → Install Android SDK

Can't find "New → Image Asset"?
-------------------------------
→ Make sure you right-click "res" folder
→ Not other folders

Icon doesn't show after install?
--------------------------------
→ Uninstall completely
→ Restart device (important!)
→ Clear launcher cache
→ Install again
