============================================
 üêõ TROUBLESHOOTING
============================================

‚ùå Update prompt not showing?
   - Check you're on native APK (not web/PWA)
   - Check versionCode in build.gradle matches useUpdateCheck.ts
   - Check update.json has higher versionCode
   - Check GitHub Pages URL works

‚ùå APK download not working?
   - Check APK URL in update.json is correct
   - Check release is published (not draft)
   - Check repository is public

‚ùå JSON not loading?
   - Wait 1-2 minutes for GitHub Pages to update
   - Check URL: https://mikuare.github.io/qmaz-project-update/update.json
   - Clear browser cache
