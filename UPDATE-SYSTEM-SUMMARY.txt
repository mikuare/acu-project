============================================

1. User opens native APK
2. After 5 seconds, app checks for updates
3. Fetches: https://mikuare.github.io/qmaz-project-update/update.json
4. Compares versionCode
5. If newer version exists ‚Üí shows update prompt
6. User clicks "Download" ‚Üí opens APK in browser
7. User installs new APK
