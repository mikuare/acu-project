============================================
 TROUBLESHOOTING
============================================

‚ùå Build fails:
   - Check for syntax errors in code
   - Run: npm install
   - Try again: npm run build

‚ùå Git push fails:
   - Check internet connection
   - May need GitHub authentication
   - Try: git push origin master (instead of main)

‚ùå Vercel not deploying:
   - Check Vercel dashboard
   - Verify GitHub repository is connected
   - May need to manually trigger deploy in Vercel

‚ùå Changes not showing:
   - Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
   - Clear browser cache
   - Wait 1-3 minutes for deployment
   - Check deployment status in Vercel

‚ùå PWA install prompt not showing:
   - Only works in browsers that support PWA
   - Only shows if not already installed
   - Won't show in native APK
   - Check browser console for errors

============================================
 MANUAL DEPLOYMENT VIA VERCEL CLI (Optional)
============================================

If automatic GitHub deployment doesn't work:

1. Install Vercel CLI:
   npm i -g vercel

2. Login to Vercel:
   vercel login

3. Deploy:
   vercel --prod
